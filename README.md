## Languages

- [Uzbek](./uz)
- [Brazilian Portuguese](./pt-br)
- [English](./en)
- [French](./fr)
- [Hebrew](./he)
- [Indonesian](./id)
- [Korean](./ko)
- [Nepali](./ne)
- [Spanish](./es)

To add a new language, create a new folder using 2 character `ISO 639-1` code of that language. For example use `hi` for `Hindi` explanations.
